# 8.1 List media API

## Summary

Add `GET /api/albums/:albumId/media` with album JWT auth, Firestore query ordered by `createdAt` ascending, response `{ media: MediaListItem[] }`.

## Approach

- **Backend**: New `listMediaByCreatedAt` in `firestore.ts` using `orderBy('createdAt', 'asc')`. Route uses `requireAlbumToken`, verifies album exists (404), returns JSON.
- **Security**: Same as other album-scoped routes; token `albumId` must match `:albumId`.
- **Frontend**: Out of scope for this story.

## Files to add or change

- `shared/types.ts` — `ListMediaResponse`; align `Media.thumbnailPath` with Firestore (`string | null`).
- `backend/src/lib/firestore.ts` — `listMediaByCreatedAt`.
- `backend/src/routes/albums.ts` — `GET /:albumId/media`.

## Acceptance criteria

- [x] Response is an array of media with those fields in createdAt order
- [x] Items with null previewPath are included
