# 1.3 Scaffold backend – Implementation plan

## Summary

Scaffold backend with Node, Express, and TypeScript; build with `tsc`, run with `node dist/index.js`; create `backend/src/` layout (index.ts, config.ts, routes/, services/, middleware/, lib/); wire CORS for frontend origin and express.json().

## Approach

- **Stack:** Node + Express + TypeScript. Build: `tsc`; run: `node dist/index.js`.
- **Layout:** `backend/src/index.ts` (Express app entry), `config.ts` (port, CORS origin), and directories `routes/`, `services/`, `middleware/`, `lib/` with minimal placeholder files so the structure exists.
- **CORS:** Allow frontend origin (e.g. `http://localhost:5173` in dev via config).
- **Body parsing:** `express.json()` so JSON request bodies are parsed.
- **Port:** 3001 per work item and frontend proxy.
