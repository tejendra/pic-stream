# 3.1 Implement seed generation – Implementation plan

## Summary

Implement BIP-39–based seed generation: a word list (2048 words) and `generateSeed(): string` that returns 5 space-separated words using `crypto.randomBytes`. The function is only invoked from the POST `/api/albums` handler (to be added in 3.2).

## Approach

- **Backend**: Add `backend/src/lib/bip39-words.ts` with the official BIP-39 English word list (2048 words) as a constant array. Add `backend/src/lib/seed.ts` with `generateSeed()`: use `crypto.randomBytes(7)` for 56 bits of entropy, take five 11-bit indices (2^11 = 2048), map each to a word, return words joined by spaces. Export only `generateSeed` from the seed module; do not call it from anywhere in 3.1 (3.2 will call it from the album-creation handler).
- **Security**: Use Node `crypto.randomBytes` for cryptographically secure selection. No logging of the seed.
