# 1.2 Scaffold frontend – Implementation plan

## Summary

Scaffold frontend with Vite, React, TypeScript, and React Router; configure `/api` proxy to backend; add minimal App with route `/` and placeholder.

## Approach

- **Stack:** Vite + React + TypeScript (per technical design). Add React Router for routing.
- **Proxy:** In `vite.config.ts`, set `server.proxy['/api']` → `http://localhost:3001` so dev requests to `/api/*` hit the backend.
- **App:** Single route `/` rendering a placeholder (e.g. "Pic Stream" or "Welcome"). No auth or API calls in this work item.
- **Security:** None required for scaffolding (static placeholder only).
